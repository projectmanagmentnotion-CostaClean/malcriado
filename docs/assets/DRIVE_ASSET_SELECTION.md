# Drive asset selection

## Criterio

A = produccion prioritaria; B = apoyo; C = reserva editorial; D = descartar.

| Grupo                            | Nota                                          | Clasificacion            | Uso                              |
| -------------------------------- | --------------------------------------------- | ------------------------ | -------------------------------- |
| Malcriado platos con fondo negro | Marca fuerte, crops utiles, parte ya presente | A/B                      | Home, Carta, Especiales          |
| malcriado 2 sushi/cocteles       | Color y detalle consistentes                  | A/B                      | Carta y escenas gastronomicas    |
| DSC preparacion/personas         | Autentico, pero contexto food truck           | C                        | Solo relato de evento confirmado |
| DSC exterior food truck          | No representa el local de Pineda              | D para Nosotros/Contacto | Archivo documental               |
| Repeticiones casi identicas      | Poco valor narrativo                          | C/D                      | No duplicar                      |
| Videos originales                | Peso incompatible sin pipeline                | C                        | Evaluacion futura                |

La revision actual prioriza crops y composicion sobre sustitucion. No se inventa una fotografia de chef, sala o terraza a partir de la serie DSC.
