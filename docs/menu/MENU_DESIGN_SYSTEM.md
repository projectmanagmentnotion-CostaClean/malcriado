# Menu Design System

## Direccion

La carta evita el patron SaaS de tarjetas repetidas. La pagina se organiza como capitulos gastronomicos con:

- hero editorial full width
- rail de navegacion por capitulos
- featured media puntual por categoria
- divisores lineales en listas
- tipografia de alto contraste
- CTAs contextuales sin sobrecargar cada plato

## Reglas visuales

- no usar sombras elevadas sistematicas
- no encapsular cada plato en una card
- usar imagen grande solo cuando mejora el ritmo editorial
- mantener lectura rapida con listas y metadatos adyacentes
- priorizar una mano en mobile para rail, CTA y retorno visual

## Estados honestos

- precio pendiente con copy explicito
- alergeno pendiente con copy explicito
- ausencia de oferta tratada como estado disenado
- `Vermut` oculto hasta recuperar contenido
